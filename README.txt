FOR LINUX

Enter the following commands in order : 

	(1) chmod +x setup_env.sh
	(2) ./setup_env.sh
	(3) source ~/myenv/bin/activate
	(4) python3 recon_tool.py



FOR WINDOWS

Save the Script: 
	Save your Python script with a .py extension, for example, script.py.

Create a Shortcut (Optional): 
	If you want a shortcut, right-click on the .py file, select "Create shortcut," and place the shortcut on your desktop.

Double-Click to Run: 
	Double-clicking a .py file should open it with Python if Python is installed correctly. Windows will use the default Python interpreter associated with .py files to run the script.
